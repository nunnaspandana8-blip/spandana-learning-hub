# Spandana Learning Hub

## Project Description
Spandana Learning Hub is a simple educational website developed using HTML, CSS and JavaScript. It provides information about courses, learning resources, and student activities in an easy-to-use interface.

## Features
- Home Page
- About Page
- Courses Page
- Gallery Page
- Contact Page
- Responsive Design
- SEO Optimized
- Live Chat Support (Tawk.to)

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Website
https://bejewelled-bubblegum-bff186.netlify.app/

## How to Run
1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any web browser.
